Como excluir seus dados

Para excluir todos os seus dados do aplicativo SageMoney, siga os passos abaixo:

  1. Abra o aplicativo SageMoney.
  2. Acesse o menu de configurações.
  3. Selecione a opção “Excluir conta”.
  4. Confirme a exclusão. Todos os seus dados serão permanentemente apagados.
